Google Drive Sync

Configure uma vez e seus dados ficam automaticamente sincronizados com o Google Drive — acessíveis de qualquer computador ou pessoa que você autorizar.

Como configurar (3 passos, ~3 min)
1
Criar projeto no Google Cloud
Acesse console.cloud.google.com → New Project → dê um nome qualquer (ex: "Finflow").
2
Ativar Google Drive API + criar credencial OAuth
Menu lateral → APIs & Services → Library → busque "Google Drive API" → Enable.
Depois → Credentials → Create Credentials → OAuth Client ID → tipo Web application.
Em Authorized JavaScript origins adicione a URL onde o arquivo será aberto (ex: file:// ou o domínio se hospedar online).
3
Cole seu Client ID abaixo
Copie o Client ID gerado (formato: XXXX.apps.googleusercontent.com)
